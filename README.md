# HR Matrix Backend

HR Matrix Backend is a RESTful API built with Node.js, Express.js, and MongoDB that powers the HR Matrix Human Resource Management System.

## Features

- User Authentication & Authorization (JWT)
- Employee Management
- Department Management
- Attendance Tracking
- Leave Management
- Payroll Management
- Recruitment & Resume Screening
- Performance Management
- Role-Based Access Control (Admin, HR, Manager, Employee)
- Secure API Endpoints
- File Upload Support

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer
- bcryptjs
